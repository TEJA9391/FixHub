# FixHub 🛠️

FixHub is a premium, high-fidelity Progressive Web App (PWA) designed to provide on-demand home services like electrical repairs, plumbing, appliance repair, and more. Built with React and Tailwind CSS, it features an Apple-inspired glassmorphic UI, fluid animations, and advanced mock functionalities.

## ✨ Features
- **Progressive Web App (PWA)**: Installable on mobile devices with native-like performance and splash screens.
- **Auto-Pilot Diagnostics**: A simulated AI-driven predictive maintenance dashboard.
- **Remote Diagnostics**: A simulated peer-to-peer Video KYC style interface with AR tool overlays for remote troubleshooting.
- **AI Chatbot**: Context-aware conversational booking system.
- **Real-Time Order Tracking**: Dynamic timeline tracking with map integrations and post-service feedback collection.
- **Secure Checkout**: Multi-step checkout with address validation and mock payment gateways.

## 🚀 Technologies Used
- **Frontend**: React.js, Vite
- **Styling**: Tailwind CSS, Framer Motion
- **Icons**: Lucide React, Google Material Symbols

## 📦 Running Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
